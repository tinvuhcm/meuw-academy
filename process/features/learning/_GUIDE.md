# learning

## Phạm Vi

Luồng học cốt lõi của app hướng tới trẻ em: từ dashboard sang session, rồi lesson, rồi hoàn tất, bao gồm tiến trình học, practice mode, và hành vi mở ngày học. Đây là lane runtime chính đang thực thi luật "dạy trước rồi luyện" trong trải nghiệm sản phẩm.
