# dashboard-settings

## Phạm Vi

Bề mặt điều khiển ngoài lesson: dashboard tổng quan, quản lý hồ sơ, khu vực PIN phụ huynh, settings, đăng nhập tài khoản, và các control đồng bộ. Feature này sở hữu các màn hình chuẩn bị, giám sát, hoặc bảo vệ trải nghiệm học, thay vì trực tiếp là luồng bài học.
