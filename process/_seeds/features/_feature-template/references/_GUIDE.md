# References

This folder holds research outputs and reference documents for this feature.

## What Goes Here

- Research outputs from RESEARCH mode (library comparisons, architecture studies)
- Reference documents that inform plans for this feature
- External documentation snapshots or summaries
- Design decisions and rationale documents (ADRs)

## Naming Convention

Use descriptive names: `{topic}-reference.md` or `{topic}-research_{dd-mm-yy}.md`

Examples:
- `auth-library-comparison-reference.md`
- `websocket-patterns-research_15-03-26.md`

## When to Create a Reference

- During RESEARCH mode when findings will inform feature plans
- When comparing implementation approaches
- When capturing external knowledge for team reference
